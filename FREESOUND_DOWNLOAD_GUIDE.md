# Freesound.org Download Guide

This document lists the recommended sounds from freesound.org for each game event. Since automated download requires authentication, these can be manually downloaded.

## How to Download

1. Visit each Freesound URL below
2. Create a free Freesound account if you don't have one
3. Click the download button on each sound page
4. Save the file with the specified filename
5. Convert to WAV format if needed (preferably mono, 22050 Hz, 16-bit)
6. Place in `composeApp/src/commonMain/composeResources/files/sounds/`

## Sound Requirements

All sounds should be:
- **Format**: WAV
- **Sample Rate**: 22050 Hz (preferred) or 44100 Hz
- **Channels**: Mono (preferred for smaller size) or Stereo
- **Bit Depth**: 16-bit
- **Duration**: Keep short (0.1 - 1.0 seconds)
- **License**: CC0, CC-BY, or other permissive license

## Download Instructions

### Manual Download
1. Click on each URL above
2. Sign in to Freesound (create free account if needed)
3. Click the "Download" button on the sound page
4. Check the license and attribution requirements

### Audio Conversion (if needed)
If the downloaded sound is not in the correct format, use a tool like:

**FFmpeg** (command line):
```bash
# Convert to mono WAV at 22050 Hz
ffmpeg -i input.wav -ac 1 -ar 22050 output.wav
```

**Audacity** (GUI):
1. Open the sound file
2. Tracks → Resample → Set to 22050 Hz
3. Tracks → Mix → Mix Stereo Down to Mono (if stereo)
4. File → Export → Export as WAV (16-bit PCM)

**Sox** (command line):
```bash
# Convert to required format
sox input.wav -r 22050 -c 1 -b 16 output.wav
```

## License Compliance

Freesound uses various Creative Commons licenses:
- **CC0**: Public domain, no attribution required
- **CC-BY**: Attribution required (add to credits)
- **CC-BY-NC**: Non-commercial use only (check game license)

After downloading, create a `SOUND_CREDITS.md` file listing:
- Sound filename
- Original author/uploader
- Freesound URL
- License type
- Any attribution text required

## Automation Option

If you have a Freesound API key, you can automate downloads:

```bash
# Example API usage (requires API key)
API_KEY="your_api_key_here"
SOUND_ID="920"

curl -H "Authorization: Token $API_KEY" \
     "https://freesound.org/apiv2/sounds/$SOUND_ID/download/" \
     -L -o sound.wav
```

To get an API key:
1. Go to https://freesound.org/apiv2/apply/
2. Fill out the application form
3. Use the provided API key in your scripts

## Search Tips

If the suggested sounds don't work well, try these search terms on Freesound:

- **Medieval sounds**: "sword", "medieval", "knight", "castle"
- **Magic sounds**: "magic", "spell", "arcane", "fireball"
- **Monster sounds**: "creature", "beast", "growl", "roar"
- **Game UI**: "game", "ui", "interface", "click"
- **Success sounds**: "success", "achievement", "level up", "win"
- **Failure sounds**: "fail", "lose", "negative", "wrong"

Always filter by:
- Duration: < 5 seconds
- License: CC0 or CC-BY preferred
- Format: WAV preferred (but convertible from other formats)
