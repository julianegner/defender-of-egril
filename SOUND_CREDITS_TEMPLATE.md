## License Types Reference

### CC0 (Public Domain)
- No attribution required (but appreciated)
- Can be used for any purpose
- No modifications restrictions

### CC-BY 3.0 / CC-BY 4.0 (Attribution)
- **Attribution required**: Must credit the author
- Can be used for any purpose including commercial
- Can be modified
- **Attribution format**: "Sound by [Author] from Freesound.org (CC-BY [version])"

### CC-BY-NC (Attribution Non-Commercial)
- Attribution required
- Non-commercial use only
- Can be modified
- Check if compatible with your project's license

### Other Licenses
- Read the specific license terms on the Freesound page
- Some sounds may have sampling+ licenses or other restrictions

## Attribution Text for Game Credits

Based on the licenses, include appropriate attribution in your game's credits:

### If all sounds are CC0:
```
Sound effects sourced from Freesound.org
```

### If sounds include CC-BY licenses:
```
Sound Effects:
- [Sound name]: [Author] from Freesound.org (CC-BY [version])
- [Sound name]: [Author] from Freesound.org (CC-BY [version])
[...etc for each CC-BY sound]
```

### Minimal attribution example:
```
Sound effects by various artists from Freesound.org under Creative Commons licenses.
Full attribution: See SOUND_CREDITS.md
```
