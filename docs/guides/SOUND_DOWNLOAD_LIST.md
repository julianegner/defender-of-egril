# Sound File Download List

This document provides direct download URLs for all 19 game sound files from Freesound.org, along with attribution information.

## Download Instructions

1. Click on each "Download Page" URL below
2. Sign in to Freesound.org (create a free account if needed)
3. Click the **Download** button on the sound page
4. Save the file with the exact filename specified
5. If the file is not WAV format, convert it (instructions at bottom)
6. Place all files in: `composeApp/src/commonMain/composeResources/files/sounds/`

## After Downloading

### 1. Check License Information

For each sound you download, note the license on the download page:
- **CC0**: Public domain, no attribution required (but appreciated)
- **CC-BY 3.0** or **CC-BY 4.0**: Attribution required
- **CC-BY-NC**: Non-commercial use only

### 2. Record Attribution Information

Create a file called `SOUND_CREDITS.md` in the repository root with the following format:

```markdown
# Sound Attribution

This game uses sounds from Freesound.org under Creative Commons licenses.

## Sound Credits

### attack_melee.wav
- **Author**: [Author name from download page]
- **Source**: [Freesound URL]
- **License**: [License type]
- **Original Filename**: [Original filename]

[Repeat for each sound...]
```

### 3. Convert Audio Format (if needed)

All sounds must be:
- **Format**: WAV
- **Sample Rate**: 22050 Hz (preferred) or 44100 Hz  
- **Channels**: Mono (preferred, smaller size)
- **Bit Depth**: 16-bit

**Using FFmpeg:**
```bash
# Convert any audio file to required format
ffmpeg -i input.mp3 -ar 22050 -ac 1 -sample_fmt s16 output.wav

# Batch convert all files in a directory
for file in *.mp3; do
    ffmpeg -i "$file" -ar 22050 -ac 1 -sample_fmt s16 "${file%.mp3}.wav"
done
```

**Using Sox:**
```bash
# Convert to required format
sox input.wav -r 22050 -c 1 -b 16 output.wav

# Batch convert
for file in *.wav; do
    sox "$file" -r 22050 -c 1 -b 16 "converted_$file"
done
```

**Using Audacity (GUI):**
1. Open the sound file
2. Tracks → Resample → Set to 22050 Hz
3. Tracks → Mix → Mix Stereo Down to Mono (if stereo)
4. File → Export → Export as WAV
5. Choose "Signed 16-bit PCM" encoding

### 4. Trim Duration (if needed)

Sounds should be short (0.1 - 1.0 seconds):

**Using FFmpeg:**
```bash
# Trim to first 0.5 seconds
ffmpeg -i input.wav -t 0.5 -c copy output.wav

# Fade out the last 0.1 seconds
ffmpeg -i input.wav -af "afade=t=out:st=0.4:d=0.1" output.wav
```

**Using Audacity:**
1. Select the portion you want to keep
2. Edit → Remove Audio or Trim Audio
3. Add fade out: Effect → Fade Out

### 5. Adjust Volume (if needed)

If sounds are too loud or too quiet:

**Using FFmpeg:**
```bash
# Decrease volume by 3dB
ffmpeg -i input.wav -af "volume=-3dB" output.wav

# Increase volume by 3dB
ffmpeg -i input.wav -af "volume=3dB" output.wav

# Normalize audio (recommended)
ffmpeg -i input.wav -af "loudnorm" output.wav
```

**Using Sox:**
```bash
# Normalize to -3dB peak
sox input.wav output.wav norm -3
```
