# Keyboard Shortcuts

This document lists all keyboard shortcuts and usable keys in Defender of Egril.

> **Note:** Keyboard shortcuts are available on **desktop** and **web/WASM** platforms only. They are not available on mobile (Android/iOS).

---

## Gameplay Screen

| Key | Action | Notes |
|-----|--------|-------|
| `F` | Attack | Fires the selected tower at the currently selected target (enemy or position). Only works during Player Turn when a tower with actions remaining is selected and a target is chosen. |
| `Ctrl+A` | Auto-attack & End Turn | Automatically fires all towers that have targets in range, then ends the turn. Only available during Player Turn. |
| `C` | Open Cheat Dialog | Opens the cheat code input dialog. |
| `E` | Toggle Enemy List | Shows or hides the enemy list overlay. |
| `Enter` | End Turn / Start Battle | Ends the current player turn. During the initial building phase, starts the first battle instead. May show a confirmation dialog if towers have unused actions. |
| `Ctrl+S` | Save Game | Opens the save game dialog. |
| `W` or `↑` | Pan Map Up | Scrolls the map upward. |
| `S` or `↓` | Pan Map Down | Scrolls the map downward. |
| `A` or `←` | Pan Map Left | Scrolls the map to the left. |
| `D` or `→` | Pan Map Right | Scrolls the map to the right. |

## Cheat Codes

Cheat codes are entered via the Cheat Dialog. Available codes:

### Gameplay
| Code | Effect |
|------|--------|
| `cash` | Add 1,000 coins |
| `mmmoney` | Add 1,000,000 coins |
| `spawn <type> <level>` | Spawn an enemy (e.g. `spawn goblin 3`) |
| `dig <outcome>` | Force next mine dig outcome (e.g. `dig gold`) |
| `platform` | Show platform information |

Valid enemy types: `goblin`, `ork`, `ogre`, `skeleton`, `wizard`, `witch`, `bluedemon`, `reddemon`, `evilmage`, `redwitch`, `greenwitch`, `ewhad`, `dragon`

### World Map
| Code | Effect |
|------|--------|
| `unlock` / `unlockall` | Unlock all levels |
| `lock` / `lockall` | Lock all levels |
| `unlock <levelId>` | Unlock a specific level |
| `lock <levelId>` | Lock a specific level |
| `sticker` | Navigate to sticker screen |
| `spinner` | Show loading spinner demo |
| `platform` | Show platform information |
