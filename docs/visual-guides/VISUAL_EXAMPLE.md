## Icon Key

### Tower Icons (All have trapezoid base with battlements)

- **Spike Tower**: `в•”в•җв•— в•‘в–Ів–Ів–Ів•‘` - Three upward spikes in yellow
- **Spear Tower**: `в•”в•җв•— в•‘ вҶ‘ в•‘` - Vertical spear with silver head
- **Bow Tower**: `в•”в•җв•— в•‘вҢ’вҶ’ в•‘` - Curved bow with arrow
- **Wizard Tower**: `в•”в•җв•— в•‘ вҳ… в•‘` - Gold star (magic symbol)
- **Alchemy Tower**: `в•”в•җв•— в•‘ вҡ— в•‘` - Green potion flask
- **Ballista Tower**: `в•”в•җв•— в•‘в•җвҶ’ в•‘` - Crossbow with bolt

### Enemy Icons

- **Goblin**: `@` with pointy ears - Small green creature
- **Ork**: `Оҳ` with tusks - Large olive green warrior
- **Ogre**: `в—ү` with big eyes - Huge brown brute
- **Skeleton**: `вҳ ` - White skull and crossbones
- **Evil Wizard**: `ОЁ` with hat - Purple wizard with staff
- **Witch**: `О»` with hat - Green witch with broom

## Color Coding

### Tower Background Colors:
- **Blue (#2196F3)**: Tower is ready with actions
- **Gray (#9E9E9E)**: Tower is being built
- **Blue-gray (#7986CB)**: Tower has no actions remaining

### Enemy Background Colors:
- **Red (#F44336)**: All enemy units

### Build Area Colors:
- **Light Green (#8BC34A)**: Build islands
- **Medium Green (#A5D6A7)**: Build strips
- **Cream (#FFF8DC)**: Enemy path
- **Gray (#E0E0E0)**: Non-playable area

## Status Overlays

All icons maintain the critical game information overlays:

- **Towers**: 
  - Level: `L1`, `L2`, `L3`, etc. (white text, bottom)
  - Actions: `вҡЎ1` (yellow, when available)
  - Build time: `вҸұ2` (orange, when building)

- **Enemies**:
  - Health: `15/20` (white text, bottom)

## Grid Cell Size

Each cell is **48dp Г— 48dp**, which provides enough space for:
1. The icon graphic (main area)
2. Status text overlays (bottom 12dp)
3. Borders for selection/highlighting (2-5dp)

The icons are designed to be clear and distinguishable even at this compact size.
